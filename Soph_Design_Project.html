Generalized Rover Project

The goal of project 1 of a 2 part project based around the control of a Dagu 5 Rover. The first project involved creating a general purpose processor in order to control the rovers movements. The rovers task was to move to a random point in a 30 inch by 30 inch grid.

Read more about it in the PDF Report for Project 1 below!

The second project was to navigate a maze using the Dagu 5. Using an Atmega 328 as the main processor and a QTR light sensor the rover moved along a white line until it reached the end of the maze.
